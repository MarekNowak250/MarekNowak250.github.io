Projekty

Komunikator sieciowy demo (BLAZOR + REST API + WebSockets)

https://gitlab.com/Phoenix510/rest_api-blazor_client-demo

Pozwala wysyłać i odbierać prywatne wiadomości pomiędzy dwoma zarejestrowanymi użytkownikami i publiczne wiadomości widoczne dla wszystkich. Wiadomości przesyłane są poprez protokuł HTTP, ale wszystkie informacje (np. o tym że dostałeś nową wiadomość) wykorzystują protokuł WebSockets.

Komunicator sieciowy (JS/WebSockets)

https://gitlab.com/Phoenix510/js_websockets

Prosta aplikacja zbudowana z wykorzystaniem JavaScript i WebSockers. Pozwala tworzyć oraz dołączać do pokoi, a także wysyłać wiadomości.

Custom Youtube Playback Speed

https://chrome.google.com/webstore/detail/custom-youtube-playback-s/mlnghacnjjppjfbbjbpkgcemiaglbpii

Rozszerzenie do przeglądarki Chrome pozwalające zmieniać prędkość odtwarzanie wideo w serwisie youtube w zakresie od 0 do 10, poprzez określony przez użytkownika krok. Kontrolki są zintegrowane z interfejsem serwisu w formie: suwaka znajdującego się pod opcją jakość w ustawieniach odtwarzacza oraz poprzez naciśnięcie określonych przez użytkownika klawiszy (domyślne '-' lub 'a' żeby zmniejszyć prędkość i '+' albo 'd' żeby zwięszyć).
